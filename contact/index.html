Instructions

  • Tirez : Maintenez le clic gauche et relâchez pour tirer la lettre dans le champ 'Courriel'. Les autres champs se remplissent normalement.
  • Durée : La touche reste immobile 1 seconde après le tir.
  • Sélectionnez : Choisissez la lettre via la roulette. Utilisez la bombe en cas d'erreur !
  • Son : Activez le son pour une meilleure expérience. Attention à ne pas trop élever le volume.
  • Envoyez : Remplissez tout le formulaire avant de cliquer sur 'Envoyer'.
